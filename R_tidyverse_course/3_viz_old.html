GGplot intro

GGplot builds figures in layers based on dataframes. We can tell it what dataframe to use and then add in visual components called aesthetics, coded aes. Each aes function tells ggplot which variables you want to represent and how you want them represented.

geom_bar()

ggplot(pets, aes(x=pet))  + geom_bar( )

#We have told ggplot to use dataframe pets, to represent pet on the x axis, and then we've added a layer in which a bar chart is used.

ggplot(pets, aes(x=pet, fill=pet))  + geom_bar( color = "black", size=3, show.legend = F)

#Now, we have told it to fill the bars with colours based on the variable pet. 
#Within the bar geom, we've told it to make the outline black, and the bars size '3'. 
#We've specified to drop the fill legend for the bars because they're specified


ggplot(pets, aes(x=pet, fill=pet))  + geom_bar( color = "black", size=3, show.legend = F) + theme_light()

#Now we've changed the background theme, which we can add extra instructions to if we want to change the defaults of that theme. If we want to set a default theme within a document, we can use theme_set() function before we start making our figures.

theme_set(theme_minimal()) #I've now set minimal, which will be how the rest of the figures below look.

geom_histogram()

ggplot(pet_happy, aes(x = age)) + geom_histogram()
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

#We've told ggplot to use dataframe pet_happy, put age on the x axis, and then to add a layer which is a histogram

ggplot(pet_happy, aes(x = age)) + geom_histogram(binwidth = 1, fill="white",color="grey")

#Now we've specified binwidth (1 unit (year) each) and told it to make the bars white with grey eding, so it looks better.

geom_density()

ggplot(x_vs_y, aes(x = x)) + geom_density()

#This gives a plots using x on x axis and a layer of density plot

#Or we can put the aes data mpping instruction in the geom itself
ggplot(x_vs_y) + geom_density(aes(x = x)) + geom_density(aes(x = y))

# BUT this doesn't give us a legend. If we want to represent both like this, we need the data long so we can get a legend.

x_vs_y %>%
  gather(key=xy, value=value, x,y) %>%
  ggplot(aes(x = value, fill=xy)) + geom_density(alpha=.5)

#Now we've made the data long and piped it into ggplot, telling it the value should be plotted, with fill colour determined by the gathered identifier variable (xy)
#The alpha element in geom_desnity determines how transparent the colour is.

geom_freqpoly()

This lets you create something thats like a density plot but is more suited to more sparse data as you can specify binwidth just like in a histogram. Below we have a histogram and freqploy plot layerd together.

ggplot(pet_happy, aes(x = age)) + geom_histogram(binwidth = 1, fill="white",color="grey") +
  geom_freqpoly(binwidth=1)
